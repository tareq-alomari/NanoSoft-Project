# 🚀 نانو سوفت - مشروع التدريب (NanoSoft Project)

مشروع عصري تم بناؤه باستخدام **React** و **Tailwind CSS** لإدارة عمليات المصادقة، عرض الإعلانات، وتصنيفات الأقسام. هذا المشروع هو نسخة مطورة ومحسنة من مشروع (NanoSoft Assignment) التقليدي.

---

##  المميزات الرئيسية (Features)
- ** نظام مصادقة متكامل**: تسجيل دخول، إنشاء حساب جديد، وحماية المسارات (Protected Routes).
- ** تصميم متجاوب (Responsive Design)**: واجهة مستخدم عصرية وأنيقة تعمل بكفاءة على جميع الشاشات (Mobile, Tablet, Desktop).
- ** سلايدر إعلانات تفاعلي**: جلب الإعلانات النشطة من الـ API وعرضها بشكل متحرك جذاب.
- ** عرض الأقسام**: استعراض كافة تصنيفات المنتجات أو الخدمات مع صورها وبياناتها.
- ** ملف شخصي متطور**: عرض بيانات المستخدم مع إمكانية حذف الحساب نهائياً.
- **🌍
- دعم اللغة العربية**: الواجهة مصممة بالكامل لتدعم RTL (من اليمين إلى اليسار).

---

##  التقنيات المستخدمة (Tech Stack)
- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Navigation**: [React Router Dom](https://reactrouter.com/)
- **API Client**: [Axios](https://axios-http.com/)

---

## كيفية التشغيل (Getting Started)

### 1. المتطلبات الأساسية
تأكد من تثبيت [Node.js](https://nodejs.org/) على جهازك.

### 2. تثبيت المكتبات
قم بالدخول إلى مجلد المشروع وتشغيل الأمر التالي:
```bash
npm install
```

### 3. تشغيل المشروع (وضع التطوير)
```bash
npm run dev
```
بعد تشغيل الأمر، افتح الرابط التالي في المتصفح: `http://localhost:5173`

---

##  هيكلية المشروع (Project Structure)
- `src/pages/`: تحتوي على الصفحات الأساسية (Login, Register, Dashboard, etc).
- `src/components/`: تحتوي على المكونات القابلة لإعادة الاستخدام (Navbar, Slider).
- `src/utils/`: تحتوي على إعدادات Axios والاتصال بالـ API.
- `src/assets/`: للصور والملفات الثابتة.
