# 🎓 دليل تعلم تطوير مشروع NanoSoft بـ React

مرحباً بك! هذا الدليل سيشرح لك كيف تحول هذا المشروع من مجرد ملفات HTML بسيطة إلى تطبيق React احترافي.

---

## 1️⃣ البداية: كيف أنشأنا المشروع؟
استخدمنا أداة تسمى **Vite**. هي المسؤولة عن بناء بيئة العمل.
الأمر الذي بدأ كل شيء هو:
```bash
npm create vite@latest NanoSoftProject -- --template react
```

---

## 2️⃣ هندسة الاتصال (API)
في المجلد `src/utils/api.js` قمنا بإنشاء "نسخة" من مكتبة **Axios**.
*   **الفائدة**: نحدد الرابط الرئيسي للسيرفر مرة واحدة فقط.
*   **Interceptor**: هو "حارس" يراقب كل طلب يخرج من موقعنا، وإذا وجد "توكن" في الذاكرة، يضعه في رأس الطلب (Headers) تلقائياً.

---

## 3️⃣ الصفحات والمكونات (Structure)
في React، لا توجد "صفحات" حقيقية، بل هي "مكونات" تتبدل.
*   **`App.jsx`**: هو العقل المدبر. يستخدم `Routes` ليقرر أي صفحة تظهر للمستخدم بناءً على الرابط.
*   **`ProtectedRoute.jsx`**: مكون وظيفته التأكد من وجود "التوكن". إذا لم يجد توكن، يقوم بطرد المستخدم لصفحة تسجيل الدخول.

---

## 4️⃣ كيف تعمل الصفحات؟ (مثال: تسجيل الدخول)
افتح ملف `src/pages/LoginPage.jsx` وستجد أهم مفاهيم React:

1.  **useState**: نستخدمه لتعريف متغيرات "حية". مثلاً `[login, setLogin]`. عندما يتغير `login` باستخدام `setLogin` يقوم React بتحديث الواجهة فوراً.
2.  **handleLogin**: دالة تعمل عند الضغط على الزر. تقوم بإرسال البيانات للسيرفر وتنتظر الرد باستخدام `await`.

---

## 5️⃣ التصميم (Tailwind CSS)
بدلاً من كتابة CSS طويل، نستخدم فئات جاهزة (Classes):
*   `flex`: لجعل العناصر بجانب بعضها.
*   `rounded-3xl`: لعمل زوايا دائرية جميلة.
*   `shadow-xl`: لإضافة ظل يعطي عمقاً للتصميم.

---

## 🚀 نصيحة للتعلم:
لا تحاول فهم كل الكود مرة واحدة. افتح ملف `Dashboard.jsx` وحاول تغيير النصوص الموجودة فيه، وستلاحظ أن المتصفح يتغير فوراً. هذا سيجعلك تشعر بقوة React!
